# 华雄模拟器网页版

## 简介
这是一个三国题材的趣味模拟器网页，基于趣味问题：关羽每10分钟杀一个华雄，华佗每15分钟救一个华雄，假如有100个华雄，要多长时间杀完？

支持自定义“初始华雄数量”“关羽杀一只华雄耗时”“华佗救一只华雄耗时”，并以可视化方式展示模拟结果。页面背景和结果图片均可自定义。

## 使用方法

### 本地体验
1. 将 `web` 文件夹下所有内容（`index.html`、`sim.js`、`hero.png`、`huaxiong.png` 等图片）放在同一目录。
2. 用浏览器（推荐 Chrome/Edge）打开 `index.html` 即可。

### 在线体验
1. 将 `web` 文件夹下所有内容上传到 GitHub 仓库根目录。
2. 在仓库设置（Settings）中启用 GitHub Pages，选择 main 分支根目录。
3. 稍等片刻，访问系统生成的网址即可在线体验。

## 文件说明
- `index.html`：主网页文件
- `sim.js`：模拟器核心逻辑
- `hero.png`：全页背景图（16:9横板，建议包含关羽和华佗形象）
- `huaxiong.png`：模拟结果展示用华雄形象

## 自定义图片
- hero.png 建议 16:9 横板，分辨率建议 1920x1080 或更高
- huaxiong.png 建议为华雄Q版形象，支持透明背景

## 体验效果
- 支持输入参数自定义
- 结果区展示华雄图片
- 页面背景为自定义大图，内容区半透明
